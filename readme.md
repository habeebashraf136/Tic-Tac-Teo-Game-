## A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. This project is perfect for beginners who want to practice DOM manipulation, game logic, and responsive UI design.


## Features
# Two-player gameplay
# Clean and responsive UI
# Winning combination detection
# Draw game detection
# Restart game button
# Easy-to-understand JavaScript logic

## Tech Stack
# HTML – Structure
# CSS – Styling
# JavaScript – Game log

## How to Play
# Player O starts the game.
# if player O or X filled the box correct then player O or X is winner
# If all cells are filled and no one wins → the game ends in a draw.
# Click Restart to play again.
